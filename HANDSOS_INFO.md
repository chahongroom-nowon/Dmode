### 4. 예시
```
만약 이렇게 되어있다면:
<table class="sales">
  <tr>
    <td>2025-09-29</td>
    <td>10:00</td>
    <td>홍길동</td>
    <td>커트</td>
    <td>25,000원</td>
  </tr>
</table>

셀렉터는:
table.sales tr
가 되고,
각 td는 날짜, 시간, 고객, 서비스, 금액 순서가 됩니다.
```
