# HandsOS 크롤링 테스트

## 현재 상태
1. 로그인 정보는 설정 완료 ✅
2. 로그인 자동화 코드 완료 ✅
3. 매출 데이터 파싱 코드 준비 완료 (실제 HTML 구조 확인 필요) ⏳

## 테스트 방법

### 1. 서버 실행
```bash
python server_data.py
```

### 2. 매출 페이지 접속
브라우저에서 `http://localhost:8000` 접속 후:
- 햄버거 메뉴 클릭
- "매출 확인" 클릭

### 3. 로그 확인
터미널에서 출력되는 로그 확인:
```
페이지 접속 중: https://www.handsos.com/login/login.asp?p=pc
로그인 완료
현재 페이지 URL: https://www.handsos.com/...
페이지 소스 일부: <html>...
```

### 4. 출력된 정보를 공유
다음 정보를 알려주세요:
- 현재 페이지 URL
- 페이지 소스 일부 (HTML 구조)

## 필요한 정보
handsos.com에 로그인했을 때:
1. **어디에 매출 정보가 표시되나요?**
   - 메인 페이지?
   - 별도 매출 페이지?
2. **매출 정보는 어떤 형식으로 표시되나요?**
   - 테이블 형태?
   - 리스트 형태?
   - 카드 형태?
3. **어떤 정보가 표시되나요?**
   - 날짜, 시간, 고객명, 서비스명, 금액 등
4. **페이지 HTML 구조는?**
   - F12로 개발자 도구 열기
   - 매출 데이터 부분 선택
   - HTML 코드 복사해서 보내주세요

이 정보를 주시면 바로 수정하겠습니다!
